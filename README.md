# Portafolio
Creo aplicaciones y herramientas que ayudan a las empresas a automatizar tareas, ahorrar tiempo y transformar sus datos en información útil para identificar KPIs, detectar patrones en la operación y tomar mejores decisiones. Para ello utilizo Inteligencia Artificial, Excel, Power BI, Looker Studio, Google Sheets, AppSheet y Google Apps Script, entre otras tecnologías.
